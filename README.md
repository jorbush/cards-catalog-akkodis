# Car catalog Akkodis

Deployed in: https://cars-catalog-akkodis.netlify.app/

### Python script instructions

- Python script path: 
```bash
cd scripts/
```

- Install requirements
```bash
pip install -r requirements.txt
```

- Run the script
```bash
python car_catalog.py
```

- See the available options
```bash
python car_catalog.py --help
```
